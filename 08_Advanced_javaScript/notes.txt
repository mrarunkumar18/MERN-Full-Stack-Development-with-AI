--> Object : It is a standalone entity used to store data in key value pairs.in In object we can store variables , arrays,, object, functions, other objects. In real life data also we represent in the form of key-value pairs. Objects are the collection of properties and methods. 
Example : 
Real life : 
    Name : "John",
    Age : 30,
    Job : "Engineer",
    Hobbies : ["Reading", "Writing", "Coding"],
    Address : {
        City : "New York",
        State : "NY",
        Zip : "10001"
    },
    Skills : function(){
        console.log("I am a ",this.Job);
    },
    

In Real Life Example:
    Name : "John",  --> Property
    Age : 30,  --> Property
    Job : "Engineer",  --> Property
    Hobbies : ["Reading", "Writing", "Coding"],  --> Property
    Address : {
        City : "New York",  --> Property
        State : "NY",  --> Property
        Zip : "10001"  --> Property
    },  --> Property
    Skills : function(){  --> Method
        console.log("I am a ",this.Job);
    },  --> Method


Objects : 
    - Properties : Variables inside the object.
    - Methods : Functions inside the object.

--> Ways to create objects:
    1. Object Literal
        const obj = {
            key : value,
        }
    2. Constructor
        function MyObject(){
            this.key = value;
        }
        const obj = new MyObject();
    3. Factory
        function MyObject(){
            return {
                key : value
            }
        }    


   --> call(), apply(), bind() --> These are used to invoke the function in a particular context. We can borrow properties and methods from one object to another object.  

      syntex : 
        
                      function.call(this, arg1, arg2, ...)   
                      function.apply(this, [arg1, arg2, ...])
                      function.bind(this, arg1, arg2, ...) 
    
    -->apply() : The apply() method calls a function with arguments provided as an array.
    
             example
                
                 function xyz(message,data){
                        console.log(`${message}, ${this.name} , ${data}`);
                    }
                    const person = {name :"Arun"}
                    xyz.apply(person,["Hello","!"])

                --> apply()  : we can pass multiple arguments in an array 
                          
                           xyz.apply(person,["Hello", "!"])
 
                    -> bind() : It does not execute the function immeditely . It return copy of the function with the this value

        -> *** Spread Operator and Rest Parameter (...)


            -> Spread Operator : It is used to spread or copy array or object data into new array or object.

                  eg:  

                    //array
        let arr=[2,3,4,5,6]
        let newArr=[...arr,89,90]
        console.log(newArr);
        //object
let obj = {name:"Arun", age:21}
let newObj = {...obj,age:22,city:"Mumbai"};
console.log(newObj);
