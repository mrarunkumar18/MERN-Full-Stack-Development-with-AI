Objects :

--> Ways to create objects:
    1. Object Literal
        const obj = {
            key : value,
        }
    2. Constructor
        function MyObject(){
            this.key = value;
        }
        const obj = new MyObject();
    3. Factory
        function MyObject(){
            return {
                key : value
            }
        }

--> Callback and asynchronous basics : 
    -> Callback : It is a function that is passed as an argument to another function and is executed after some operation is completed. It is used to handle asynchronous operations in JavaScript. 

    -> Asynchronous : It is a programming paradigm that allows multiple operations to be performed concurrently without blocking the main thread. In JavaScript, we can use callbacks, promises, and async/await to handle asynchronous operations.
       

       eg: 
       console.log("Start");
       setTimeout(()=>{
        console.log("This is a callback function");
       },2000);
       console.log("End");


       -> callbacks : are used to perform asynchronous operations in javascript. It is a function which is passed as argument to another function and is executed after some operation is completed.

       eg: 
         function xyz(callback){
            setTimeout(()=>{
