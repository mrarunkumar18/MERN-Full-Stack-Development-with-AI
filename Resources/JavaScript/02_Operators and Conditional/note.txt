--> Arithmetic Operators: Used for mathematical operations.

        +, -, *, /, %, ** 

        + : Addition or Concatenation
        - : Subtraction
        * : Multiplication
        / : Division
        % : Modulus
        ** : Exponentiation



     * Moudulus --> Returns the remainder of the division. (Ex : 5 % 2 = 1)
     * Exponentiation --> Returns the result of raising the first operand to the power of the second operand. (Ex : 5 ** 2 = 25)
     * Increment --> Increments the value of the operand by 1. (Ex : 5++ = 6)
     * Decrement --> Decrements the value of the operand by 1. (Ex : 5-- = 4)

    Assignment Operator --> Used to assign values to variables.

        = : Assignment
        += : Addition and Assignment (Ex : a += b is same as a = a + b)
        -= : Subtraction and Assignment (Ex : a -= b is same as a = a - b)
        *= : Multiplication and Assignment (Ex : a *= b is same as a = a * b)
        /= : Division and Assignment (Ex : a /= b is same as a = a / b)
        %= : Modulus and Assignment (Ex : a %= b is same as a = a % b)
        **= : Exponentiation and Assignment (Ex : a **= b is same as a = a ** b)

    Comparison Operator --> Used to compare two values.

        == : Equal to
        != : Not equal to
        === : Strictly equal to
        !== : Strictly not equal to
        > : Greater than
        < : Less than
        >= : Greater than or equal to
        <= : Less than or equal to

    Logical Operator --> Used to combine two or more conditions.

        && : Logical AND (Returns true if both conditions are true)
        || : Logical OR (Returns true if at least one of the conditions is true)
        ! : Logical NOT (Returns the opposite of the condition)

    Conditional --> used to make decisions in the code. 

    1. If Statement 
        if (condition) {
            // code to be executed if the condition is true
        }

    2. If-Else Statement 
        if (condition) {
            // code to be executed if the condition is true
        } else {
            // code to be executed if the condition is false
        }

    3. If-Else-If Statement 
        if (condition1) {
            // code to be executed if condition1 is true
        } else if (condition2) {
            // code to be executed if condition2 is true
        } else {
            // code to be executed if both conditions are false
        }

    4. Nested If Statement 
        if (condition1) {
            if (condition2) {
                // code to be executed if both conditions are true
            } else {
                // code to be executed if condition1 is true and condition2 is false
            }
        } else {
            // code to be executed if condition1 is false
        }


        Ternary Operator --> Used to make decisions in the code. (Ex : condition ? value1 : value2) 


        let age = 18;
        let result = age >= 18 ? "Adult" : "Not Adult";
        console.log(result);
