Loops and Strings  

 #Loops : Loops are used to execute a block of code multiple times. 
           
 #Types of Loops: 
             1. For Loop
             2. While Loop 
             3. Do While Loop
             4. For-of Loop
             5. For-in Loop 

        1. For Loop: 

            Syntax: 
                    for (initialization; condition; increment/decrement) { 
                        // code to be executed 
                    } 


            Example: 
                   for (let i = 0; i < 10; i++) { 
                        console.log(i); 
                   } 
            
                * Infinite Loop: 
                        for (let i=1; i<=5; i--) {    // Error: Infinite Loop: Condition will always be true 
                            console.log("Hello"); 
                        }
                
                * Block scope: 
                        for (let i = 1; i <= 5; i++) { 
                            sum = sum + i; 
                        }
                        console.log(sum); // Error: sum is not defined 

        2. while Loop : 
                Syntax: 
                        while (condition) { 
                            // code to be executed 
                        }
                
                Example: 
                        let i = 0; 
                        while (i < 10) { 
                            console.log(i); 
                            i++; 
                        }
                
                * Infinite Loop: 
                        while (true) { 
                            console.log("Hello"); 
                        }
                
                * Block scope: 
                        while (i < 10) { 
                            let sum = 0; 
                            sum = sum + i; 
                        }
                        console.log(sum); // Error: sum is not defined 
            

        3. Do While Loop: 
                Syntax: 
                        do {
                            // code to be executed
                        } while (condition); 
                
                Example: 
                        let i = 0; 
                        do { 
                            console.log(i); 
                            i++; 
                        } while (i < 10); 
            
        4. For-of Loop: 
                Syntax: 
                        for (let variable of iterable) { 
                            // code to be executed 
                        } 
                
                Example: 
                        let str = "Hello"; 
                        for (let char of str) { 
                            console.log(char); 
                        } 
            
        5. For-in Loop: 
                Syntax: 
                        for (let key in object) { 
                            // code to be executed 
                        } 
                
                Example: 
                        let obj = { name: "John", age: 30, city: "New York" }; 
                        for (let key in obj) { 
                            console.log(key + ": " + obj[key]); 
                        } 

        6. Nested Loop: 
                Syntax: 
                        for (let i = 0; i < 5; i++) { // Outer Loop 
                            for (let j = 0; j < 5; j++) { // Inner Loop 
                                // code to be executed 
                            } 
                        }

 #Strings: Strings are sequences of characters that are used to represent text. 

        Example: 
               let str = "Hello"; 
               console.log(str); 

        #String Properties: 
               let str = "Hello"; 
               console.log(str.length); // Length of the string 

       *Templates Literals: A way to create strings with embedded expressions.
               `this is a template literal ${}`  

       *String Interpolation: A way to create strings with embedded expressions.
               `this is a template literal ${2+2}`

       *String Methods:
               let str = "Hello";
               console.log(str.toUpperCase()); // Convert string to uppercase
               console.log(str.toLowerCase()); // Convert string to lowercase
               console.log(str.trim()); // Remove whitespace from string
               console.log(str.slice(1, 3)); // Extract a section of string
               console.log(str.substring(1, 3)); // Extract a section of string
               console.log(str.replace("Hello", "World")); // Replace a string
               console.log(str.split("")); // Split a string into an array
