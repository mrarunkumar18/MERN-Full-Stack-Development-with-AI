Arrays in JavaScript

#Arrays : Arrays are a collection of items stored at contiguous memory locations (in many languages, though in JS they are dynamic objects). They are used to store multiple values in a single variable.

#Syntax : 
        let arr = [element1, element2, element3];

#Note : 
        - Arrays are 0-indexed (first element is at index 0).
        - typeof(array) is "object".
        - Arrays can store different types of data (number, string, boolean, object, etc.).

#Array Properties :
        - length : returns the number of elements in an array.
          Example: arr.length;

#Looping over Arrays :
        1. For Loop:
           for(let i=0; i<arr.length; i++) {
               console.log(arr[i]);
           }

        2. For-of Loop:
           for(let element of arr) {
               console.log(element);
           }

        3. forEach Loop:
           arr.forEach((val, index) => {
               console.log(val, index);
           });

#Array Methods :
        1. push() : adds element to the end of the array. (modifies original)
        2. pop() : removes element from the end and returns it. (modifies original)
        3. toString() : converts array to string.
        4. concat() : joins multiple arrays and returns result.
        5. unshift() : adds element to the start.
        6. shift() : removes element from the start and returns it.
        7. slice(start, end) : returns a piece of array. (does not modify original)
        8. splice(start, deleteCount, newElements) : change contents of array. (modifies original)

#Higher Order Array Methods (Advanced) :
        1. map() : creates a new array by performing some operation on each element.
        2. filter() : creates a new array with elements that pass a test.
        3. reduce() : reduces array to a single value.
