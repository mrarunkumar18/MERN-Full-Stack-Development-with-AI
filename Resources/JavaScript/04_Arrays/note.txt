Arrays in JavaScript

# Arrays: Arrays are used to store multiple values in a single variable. They are objects that can hold elements of any data type.

# Characteristics:
- Ordered: Elements are stored in a specific sequence.
- Index-based: Elements are accessed using their position (starting from 0).
- Mutable: You can change the values of elements.
- Heterogeneous: Can store different types of data (though typically used for same-type).

# Syntax:
  let heroes = ["ironman", "thor", "hulk", "spiderman"];
  let marks = [96, 75, 48, 83, 66];
  let info = ["Arun", 22, "Lucknow"]; // String, Number, String

# Accessing Elements:
  console.log(marks[0]); // 96
  marks[0] = 66; // Changing value

# Array Length:
  console.log(marks.length);

# Looping over an Array:
  1. For Loop:
     for (let i=0; i < heroes.length; i++) {
         console.log(heroes[i]);
     }

  2. For-of Loop (Recommended for values):
     for (let hero of heroes) {
         console.log(hero);
     }

# Array Methods:
  1. Push(): Add to end (returns new length)
  2. Pop(): Delete from end (returns deleted element)
  3. ToString(): Converts array to string
  4. Concat(): Joins multiple arrays & returns result (non-mutating)
  5. Unshift(): Add to start (returns new length)
  6. Shift(): Delete from start (returns deleted element)
  7. Slice(start, end): Returns a piece of the array (non-mutating, end index not included)
  8. Splice(start, deleteCount, newElements...): Changes contents of array (add, remove, replace)
     - splice(index, count, val1, val2) -> mutates original
