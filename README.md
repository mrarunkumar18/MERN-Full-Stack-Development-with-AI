# MERN Full-Stack Development with AI

Comprehensive workspace for learning and building full-stack web applications using the MERN stack (MongoDB, Express, React, Node) with progressive AI-assisted workflows and practical projects.

## Overview

This repository is a structured learning and project workspace capturing the journey from core web fundamentals (HTML, CSS, JavaScript) through modern frontend and backend development, and into full-stack projects enhanced by AI tools.

Key goals:
- Master frontend fundamentals and React
- Learn backend fundamentals with Node.js and Express
- Use MongoDB for data persistence
- Build deployable full-stack apps and incorporate AI-assisted features

## Quickstart (Local)

1. Clone the repo:

```bash
git clone https://github.com/mrarunkumar18/MERN-Full-Stack-Development-with-AI.git
cd MERN-Full-Stack-Development-with-AI
```

2. Inspect folders and open in VS Code:

```bash
code .
```

3. There are multiple learning folders (HTML/CSS/JS). For any Node/Express app you create later, run:

```bash
npm install
npm run dev   # or `node index.js` depending on project
```

Notes: This repository currently contains learning modules and frontend practice projects. When backend projects are added, they will include their own `package.json` and run instructions.

## Workspace Structure (key folders)

- `00_Foundation/` — foundational notes and quick references
- `01_Introduction to Web Basics/` — basic HTML examples and forms
- `02_Version Control & CSS Concepts/` — CSS fundamentals and example pages
- `03_CSS Layout Techniques/` — grid, layout examples, pseudo-classes
- `04_Responsive Design with Bootstrap/` — Bootstrap examples and responsive patterns
- `05_javaScript_foundatinon/` — basic JS pages and examples
- `06_Java_Script_Essentials/` — ES6 fundamentals and notes
- `07_javaScript_start/` — control flow, loops, practice problems
- `08_Advanced_javaScript/` — closures, inheritance, this-context, async basics, spread/rest
- `Projects/` — sample projects and portfolio
- `Resources/` — structured practice problems and short exercises

Each folder contains small, self-contained examples that progressively build conceptual knowledge.
