--> CSS: Cascading Style Sheets. It is used to style the HTML document - colors, font, spacing, border, etc. 
    syntex

     selector{
        property: value;
     } 
     eg: 
     h1{
        color: 'red'
     }
     -> How we add styles (types of styles)
       -> Inline Styles : These styles are define in any tag with the help of style sttribute. 
       eg: 
       <p style="color: red;">
       ---
       ---
       </P>
       -> Internal (embedded style): These styles are define in head section with the help of style tag. 
       eg: 
       <html>
           <head>
              <styel>
              h1{}
              p{}
              a{}
              </styel>
           </head>
           <body>
            
           </body>
       </html>

     -> External Styles(CSS): We can create a css file with an extension of .css  and then we call that css file in any of the webpage of my website.
        h1{}
        p{}
        a{}
        my.html
        <html>
            <head>
                <link rel="stylesheet" href="style.css">
            </head>
            <body>
            ---- 
            </body>
        </html>


--> CSS Selectors: 
 -> Universal Selectors (*) 
    *{
    styles will be applied to all the elements
    }

 -> Element Selectors (tag name)
    h1 , p , a etc 
    h1{
    styles will be applied to all the h1 elements
    }

 -> class selector
    eg:
    .error{
        color: red;
    }
    <p class="error">This is an error</p>

 -> Id Selectors : (Unique style) , it is define with hash(#) symbol with id name and call in any tag with id attribute.
    eg:
    #main{
        //style will be applied to the element with id "main"
    }
    <div id="main">
        <h1>This is a heading</h1>
        <p>This is a paragraph</p>
    </div>

 }

-> setting color property 
   body{
      color: red
   }
   -> color values
    -> color keyword or name red, green, yellow etc
    -> hex value - like #ff0000  (6 digits) 
    -> rgb value - like rgb(255, 0, 0)
    -> rgba value - like rgba(255, 0, 0, 0.5)
    -> hsl value - like hsl(0, 100%, 50%)
    -> hsla value - like hsla(0, 100%, 50%, 0.5)
     
     -> inline ----> internal ----> external 

   -> pseudo classes : it allow to style the dynamic states of an element  such as hover, active, focus state or element existing into our document.
      starting with : 
       selector : pseudo-class{
         styles
       }
       eg: 
       a:hover{
        color: red;
       }
       a:link{
        color: green;
       }
       a:visited{
        color: blue;
       }
       a:active{
        color: yellow;
       }

       -> :first-child : matches an element that is the first child of its parent.
       
   -> Pseudo-element (::) : It allow to style the specific part of an element.
      selector :: pseudo-element{
        styles
      }
      eg: 
        :: first-line
