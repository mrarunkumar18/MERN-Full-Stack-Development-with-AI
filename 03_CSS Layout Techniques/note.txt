--> CSS: Cascading Style Sheets. It is used to style the HTML document - colors, font, spacing, border, etc. 
    syntex

     selector{
        property: value;
     } 
     eg: 
     h1{
        color: 'red'
     }
     -> How we add styles (types of styles)
       -> Inline Styles : These styles are define in any tag with the help of style sttribute. 
       eg: 
       <p style="color: red;">
       ---
       ---
       </P>
       -> Internal (embedded style): These styles are define in head section with the help of style tag. 
       eg: 
       <html>
           <head>
              <styel>
              h1{}
              p{}
              a{}
              </styel>
           </head>
           <body>
            
           </body>
       </html>

     -> External Styles(CSS): We can create a css file with an extension of .css  and then we call that css file in any of the webpage of my website.
