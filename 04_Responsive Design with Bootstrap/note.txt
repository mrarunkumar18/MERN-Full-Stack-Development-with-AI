--> CSS Box Model
     -> dimensions
        width
        height 
        eg:
          div{
            width:300px;
            height:200px;

          }            

    -> padding : the space between the content and the border
                 padding-top
                 padding-bottom
                 padding-left
                 padding-right

                 eg: padding:20px;
                     padding:10px 20px;
                     padding:10px 20px 30px;
                     padding:10px 20px 30px 40px;

    -> margin : the space between the border and the next element
                 margin-top
                 margin-bottom
                 margin-left
                 margin-right

                 eg: margin:20px;
                     margin:10px 20px;
                     margin:10px 20px 30px;
                     margin:10px 20px 30px 40px;

    -> border : the line around the content and padding
                border-width
                border-style
                border-color

                eg: border:2px solid black;
                    border:1px solid red;
                    border:3px solid green;
                    border:4px solid blue;

    -> border-radius : the curve around the border
                       eg: border-radius:10px;

    -> box-shadow : the shadow around the border
                     eg: box-shadow: 2px 2px 5px black;


    -> CSS Box Sizing
        box-sizing: border-box; /* content + padding + border = total width */
        box-sizing: content-box; /* content + padding + border = extra width */

    -> positioning techniques : 
      -> static positioning : It always positioned according to the normal flow of the page. All elements static by default.
      -> relative positioning : It is positioned relative to its normal position.
      -> Absolute : It is positioned relativeto its first parent element , which have a postion relative. If no parent element have a postion relative then it is positioned relative to the body element.
      -> Fixed : It is positioned relative to the viewport.
      -> Sticky : It is positioned relative to the viewport , but it is also positioned relative to its parent element.

         

    -> Flexbox layout system (one dimensional layout): new layout model introduced in css3
    display:flex 
      at parent, then all the children of the parent are inline
