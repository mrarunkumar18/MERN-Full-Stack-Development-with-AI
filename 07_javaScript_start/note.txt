--> Javascript is a programming language that is commonly used to create interactive effects within web browsers. It allows developers to implement complex features on web pages, such as dynamic content updates, form validation, animations, and more. JavaScript can be used on both the client-side (in the browser) and the server-side (with environments like Node.js). It is an essential part of modern web development and works alongside HTML and CSS to create engaging user experiences.

   -->Variables : are temporary memory location which hold data
      -> Old javascript 
          var varname = value;
     -> Let keyword (block scoped)
     let varname=value;
       {
        let x=10;
       }     
    -> Rules to define Variables name
       1. Start with alphabate or _ or $
       2. Cannot start with number
       3. Can contain alphabate, number, _ and $
       4. after that combination of (a-z, A-Z, 0-9, _ and $)
       eg:
       var name = "John";
       let age = 30;
       const PI = 3.14;   

  --> Constant : It si a value that never be changed, it is fixed nd permanent (block scoped)
     const varname=value;
     eg:
     const PI = 3.14;


  --> Data types : type declaration of variables. Javascript is an loosely typed langauge, it means datatypes is alloated at runtime as per the variabble value. 
  eg;
    let name = "John"; // String
    let age = 30; // Number
    let isStudent = true; // Boolean
    let hobbies = ["reading", "gaming", "coding"]; // Array
    let person = { name: "John", age: 30 }; // Object
    let undefinedVariable; // Undefined
    let nullVariable = null; // Null

  --> Operators to perform operations 
        1. Arithmetic Operators : +, -, *, /, %, ++, --
                + : it is used for addition
                - : it is used for subtraction
                * : it is used for multiplication
                / : it is used for division
                % : it is used to find the remainder
                ++ : it is used to increment a variable by 1
                -- : it is used to decrement a variable by 1

        2. Assignment Operators : =, +=, -=, *=, /= 
                = : it is used to assign a value to a variable
                += : it adds the right operand to the left operand and assigns the result to the left operand
                -= : it subtracts the right operand from the left operand and assigns the result to the left operand
                *= : it multiplies the left operand by the right operand and assigns the result to the left operand
                /= : it divides the left operand by the right operand and assigns the result to the left operand
                
        3. Comparison/Relational Operators : ==, ===, !=, !==, >, <, >=, <=
            == : it checks only value
            === : it checks value and datatype both
            != : it checks only value
            !== : it checks value and datatype both

        4. Logical Operators : &&, ||, !
            && : it returns true if both the operands are true
            || : it returns true if at least one of the operands is true
            ! : it returns true if the operand is false and vice versa

        5. Bitwise Operators : &, |, ^, ~, <<, >>
             & : it performs bitwise AND operation
             | : it performs bitwise OR operation
             ^ : it performs bitwise XOR operation
             ~ : it performs bitwise NOT operation
             << : it performs left shift operation
             >> : it performs right shift operation

        6. Ternary Operator : condition ? expression1 : expression2
                It is a shorthand for an if-else statement. If the condition is true, expression1 is executed; otherwise, expression2 is executed.
